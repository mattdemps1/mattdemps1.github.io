Molecular Bit

The remote control for programmable therapeutics.

Molecular Bit is building remotely controlled molecular switches for wireless, reversible control of protein activity. The architecture is protein-agnostic — a command layer any therapeutic can plug into. One platform, built to be the foundation molecular robotics gets built on top of.

No off switch.

Once a drug is inside your body, you can't control it. It runs at full strength until it's gone. For a headache, that's fine. For a cancer treatment or a brain condition, it isn't.

Molecular Bit is a remote control for medicine. The protein is off by default. A physician switches it on, dials it back, or turns it off again — from outside the body, in real time.

UAV Predictive Interception

Object permanence for drones.

First-authored conference paper. Humans can close their eyes and still know where a moving object is going. Drones can't — the moment a target leaves the camera frame, it's gone. This system gives drones that same spatial memory, so they can predict where a target will be and intercept it even through blind spots.

UAV Predictive Interception — Interactive
STATE: STANDBY

Aerospace Systems Lab · Indiana University

Drones lose situational awareness the moment a target disappears behind an obstacle or exits the camera frame. The system I built treats this like a human tracking problem — predicting where the target went, not just where it was.

The framework fuses camera data with inertial motion sensors to maintain a continuous target state estimate through visual gaps. When the target reappears, the drone is already pointed in the right direction.

Computational Enzyme Switch

Built for a reaction that didn't exist in nature.

Led the computational design of a protein cap that gates an enzyme's active site — think of it like a trapdoor that opens and closes based on what's chemically present.

Catalytic Cycle — Interactive Apo state — active site open

Multi-State Protein Design

When the signal is there, the cap closes, sealing out interference and letting the reaction run cleanly. When it's gone, the cap opens and the enzyme sits idle. No natural version existed for this scaffold, so one was designed entirely from scratch using generative AI and screened computationally across both states. Ranked candidates ready for wet lab validation.

ShaderNN

What if a neural network had no runtime?

A complete neural network running entirely inside a GPU shader — no ML framework, no backend, no runtime. Every pixel on your screen is a neuron. Model weights ship as a PNG.

ShaderNN — Neural Cellular Automata · Live WebGL2

Neural Cellular Automata · WebGL2

Every device with a screen already has a GPU. ShaderNN runs inference directly in the graphics pipeline — 65,536 neurons updated in parallel every frame. The model grows target shapes from a single pixel and is self-healing: damage the pattern, it regenerates itself. Weights are ~12KB total. Training is PyTorch. Inference is pure WebGL2 — nothing else.
